# h-stick
这是一个滑动置顶,具体请看demo.

## 引入方式
1.npm

     $ npm install @h-stick --save //安装

     require('h-stick')
     或者
     import 'h-stick';



## 初始化
    hStick.init();

滑动置顶的元素添加class:sticky-wrap

    <div class="sticky-wrap">
            <ul class="nav">
                <li>推荐</li>
                <li>分类</li>
                <li>头条</li>
            </ul>
    </div>
